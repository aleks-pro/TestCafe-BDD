# TestCafe-BDD
TestCafe framework repo to troubleshoot problems to implement using CucumberJS

# Setup
- Download or clone this repo
- Run npm install

# Install Testcafe

- Run npm install Testcafe

# Run Test
- npm run test
- refer to package.json for the script
